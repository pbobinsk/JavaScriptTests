
### **Instalacja TypeScript**

```sh
npm install -g typescript
```

Sprawdzenie
```sh
tsc -v
```

### **Tworzenie projektu i pliku TypeScript**


```sh
mkdir typescript-project && cd typescript-project
```

```sh
tsc --init
```

### **Kompilacja i uruchomienie kodu**

```sh
tsc index.ts
```

```sh
node index.js
```
lub


```sh
npm install -g ts-node
ts-node index.ts
```
